== Description ==

A great amount of time is required in configuring the necessary admin settings. A single configuration change may have multiple impacts, be it minor or major. In order to make sure that the configurations are not disturbed, Admin Activity Logs for Wordpress provides a feature in the admin panel that keeps a track of all the activities being performed in the admin.
